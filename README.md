# 💻 DaudDev Personal Portfolio Website

Selamat datang di portofolio pribadi saya — *Daud Hanafi*, seorang front-end developer enthusiast dari SMK Muhammadiyah 1 Playen. Website ini dibuat sebagai media untuk memperkenalkan diri, memamerkan keterampilan coding, serta menyediakan kontak langsung melalui form email.

---

## 🚀 Fitur Utama

- ✅ Tampilan modern dan responsif (mobile-friendly)
- ✅ Formulir kontak yang langsung mengirim email via [EmailJS](https://www.emailjs.com/)
- ✅ Navigasi smooth dan user-friendly
- ✅ Menggunakan font dari Google Fonts (Inter, Poppins, Kanit)
- ✅ Desain ringan dan estetik

---

## 🛠️ Teknologi yang Digunakan

- **HTML5** – Struktur halaman
- **CSS3** – Styling dan layouting responsif
- **JavaScript** – Fungsi interaktif & integrasi EmailJS
- **EmailJS** – Pengiriman email langsung dari form
